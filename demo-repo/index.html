Centering in css

Centering in css has always been difficult. As CSS-Tricks puts it,

"Centering things in CSS is the poster child of CSS complaining. Why does it have to be so hard? They jeer. I think the issue isnt that its difficult to do, but in that there so many different ways of doing it, depending on the situation, its hard to know which to reach for."

This website will demonstrate how to center:

  • One block level element

And provide resources to learn how to center:

  • Several block level elements (via CSS-Tricks)

This form of centering is particularly useful when learning how to build smaller scale websites, as this works most effectively when paired with small and compact content

Demo

Part 1: Writing the HTML

The HTML for this website is very simple. Here are the steps to complete it:

  1. ! tab to get your normal HTML layout set
  2. name the Document
  3. link your CSS
  4. Inside of the body make a div with text of your choice (The example uses "I'm a block level element and centered")
  5. Double check it matches the document shown below

HTML:

                        
                <!DOCTYPE html>
                <html lang="en">
                            
                <head>
                <meta charset="UTF-8">
                <meta http-equiv="X-UA-Compatible" content="IE=edge">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title> Demo 2.2.22</title>
                <link rel="stylesheet" href="css/style.css">
                </head>
                
                <body>
                <div class="center">
                I'm a block level element and centered
                </div>
                </body>
                            
                </html>

When done, create a css file inside of a css folder which is named to whatever you had linked in your HTML doc

Part 2: CSS

This is the CSS for the previously shown HTML

There are no clear instructions for this section, as CSS can be freer with colors, themes, etc.

However, there are some requirements, which are:

  1. colors of some kind are shown in each section
  2. Your body has font-size 80%
  3. The parts for inline-block-center, inline-block-center div, and flex-center

CSS:

                        
                            body {
                                background: orange;
                                font-size: 80%;
                            }
                            
                            main {
                                background: white;
                                margin: 20px 0;
                                padding: 20px;
                            }
                            
                            main div {
                                background: black;
                                color: white;
                                padding: 15px;
                                max-width: 125px;
                                margin: 5px;
                            }
                            
                            .inline-block-center {
                                text-align: center;
                            }
                            
                            .inline-block-center div {
                                display: inline-block;
                                text-align: left;
                            }
            
                            .flex-center {
                                display: flex;
                                justify-content: center;
                            }

If you wish to expand on this skill more, use this link to find a further demo which uses multiple elements

At the end, you should have a website that meets the requirements somewhat, but there is not a lot of set requirements. So long as the mastery of centering is shown it's good
